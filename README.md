# New Features !

  - Add ARMv8 arch

#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/docker-pcc-revived.git
```
#### Build
```sh
cd docker-pcc-revivedc && make linux/amd64
```
#### Usage

```sh
./pcc-revived.sh pcc hello.c
```
